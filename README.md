# MangaSearch
Ko gledaš spletne strani kot je Comick imajo bralci mang, ki jih berejo že dlje časa probleme pri iskanju kvalitetnih mang zaradi preslabega filtriranja. V projektu bom poskusil rešiti ta problem z delanjem bolj specifičnih filtracij.
